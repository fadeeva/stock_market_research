# Stock Market Research

**An experimental laboratory for quantitative finance, algorithmic trading, and market microstructure research.**

This repository contains my personal exploration of concepts, models, and algorithms used in systematic trading. The code is written for understanding and experimentation, not for live trading without further validation. It reflects my journey from financial theory to executable logic.
